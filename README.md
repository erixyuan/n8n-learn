# n8n-learn

n8n工作流自动化教学视频系列项目

## 项目简介

本项目旨在创建一套完整的n8n教学视频系列，从零基础入门到高级应用，帮助学习者系统性地掌握n8n工作流自动化平台。

## 教学视频规划

- **总集数**: 约20集主体内容 + 附加内容
- **每集时长**: 15-25分钟
- **难度递进**: 从基础概念到企业级应用
- **实践导向**: 每集包含实际操作演示
